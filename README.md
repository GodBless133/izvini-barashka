# 🌹 Извини барашка

Одностраничный лендинг с искренними извинениями «Извини барашка» на фоне кустовых роз.

## ✨ Особенности

- Красивый фон из AI-сгенерированных кустовых роз
- Крупный заголовок элегантным serif-шрифтом (Playfair Display)
- Милое изображение барашка среди роз
- Стеклянная карточка (glassmorphism) с мягким свечением
- Анимация падающих лепестков роз (Framer Motion)
- Интерактивная кнопка «Принять извинения» с пульсирующим сердечком
- Адаптивный дизайн (мобильные + десктоп)

## 🛠 Технологии

- [Next.js 16](https://nextjs.org/) (App Router)
- TypeScript
- Tailwind CSS 4 + shadcn/ui
- Framer Motion (анимации)

## 🚀 Запуск локально

```bash
bun install
bun run dev
```

Откройте http://localhost:3000

## 📦 Деплой на Vercel

1. Запушьте репозиторий на GitHub.
2. Зайдите на [vercel.com](https://vercel.com) → **New Project**.
3. Импортируйте репозиторий из GitHub.
4. Нажмите **Deploy** — Vercel сам определит Next.js и соберёт проект.

Готовая ссылка появится через ~30 секунд.
